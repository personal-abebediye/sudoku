# Sudoku

A full-featured cross-platform Sudoku game built with Flutter.

## Features (Planned)

- 🎮 **Core Gameplay**: Classic 9x9 Sudoku with four difficulty levels
- 💡 **Smart Hints**: Multiple hint types to help you learn
- ⏱️ **Timer & Stats**: Track your progress and completion times
- 🏆 **Achievements**: Unlock badges for your accomplishments
- 📅 **Daily Challenges**: Fresh puzzles every day
- 🎨 **Themes**: Light, dark, and custom color themes
- ☁️ **Cloud Sync**: Optional cross-device progress sync
- 📱 **Cross-Platform**: iOS, Android, Web, macOS, Windows, Linux

## Getting Started

### Prerequisites

- Flutter SDK 3.0.0 or higher
- Dart SDK 3.0.0 or higher

### Installation

```bash
# Clone the repository
git clone https://github.com/abebediye/sudoku.git
cd sudoku

# Install dependencies
flutter pub get

# Run the app
flutter run
```
